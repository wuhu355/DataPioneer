## 📖 项目简介

**DataPioneer（数据先锋大屏）** 是一个面向数据可视化初学者的开源项目，旨在帮助大家从 0 到 1 学习如何亲手制作一个炫酷的数据可视化大屏。

无论你是前端新手还是对数据可视化感兴趣的开发者，本项目都将带你一步步掌握数据大屏的核心技能——从数据准备、图表选型、布局设计到交互动效，手把手教你打造属于自己的数据大屏。

## 🎯 项目目标

- ✅ 从零开始，覆盖数据大屏开发的完整流程
- ✅ 提供清晰易懂的教程和代码示例
- ✅ 涵盖常用的数据可视化图表类型（折线图、柱状图、饼图、地图等）
- ✅ 介绍大屏适配方案（多种屏幕分辨率适配）
- ✅ 炫酷的动效和交互设计

## 🛠️ 技术栈（规划中）

- **前端框架**: React / Vue（待定）
- **可视化库**: ECharts / D3.js
- **样式方案**: CSS Modules / Tailwind CSS
- **构建工具**: Vite
